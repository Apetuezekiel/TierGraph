# tiergraph

Zero-runtime-dependency TypeScript library for SaaS tier-based feature gating.

> Under construction — v0.1.0 coming soon.

## Overview

`tiergraph` lets you declare your pricing plans and feature gates in one typed schema, then enforce them at runtime with a tiny, auditable API surface. No SDK bloat, no network calls, no framework lock-in.

Full documentation will be published with v0.1.0.
